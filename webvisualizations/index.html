    Summary: Latitude vs X

    The aim of this project is to analyze the relationship between latitude and the weather variables of temperature, humidity, cloudiness, and wind speed. The data used in this project is from the OpenWeatherMap API, which covers weather data from over 5000 cities.

    After pulling the data from the API in python, I created a dataframe to easily view the data and used Matplotlib to create graphs, illustrating the relationship between these variables. Here you can view all of the data and visualizations created, as well as analyses on the different visible trends.
